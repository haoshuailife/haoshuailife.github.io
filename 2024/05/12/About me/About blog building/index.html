本博客搭建过程

haoshuai 2024-05-12 19:04:18
Categories： Tags：

hexo blog搭建

  1. github仓库 + hexo开源项目+webstorm编译器 + nodejs支持
  2. 安装node.js,git。
    检测方法
    git –version
    node -v
    npm -v
  3. 本地开发环境，我用的webstorm，安装hexo
    命令：npm install -g hexo-cli
    切到想安装目录下，hexo init
    hexo s 查看效果 端口默认4000
  4. 建立本地项目和github仓库的联系
    这里说下为啥不部署到gitee，因为它要实名，很坑。
    • 在github新建一个仓库，这里仓库名是有要求的，需要和用户名一样，比如
      建好之后clone链接为 git@github.com:abc/abc.github.io.git,用户名为abc
      这里仓库名就要填写abc.github.io
    • 然后将git生成的公钥添加到github
    • 本地装插件 支持通过git上传 npm install hexo-deployer-git –save
      • 修改hexo配置文件 _config.yml
        ···
        deploy:
        type: git
        repo: git@github.com:abc/abc.github.io.git
        branch: master
        ···
    • hexo g 编译静态文件
    • hexo d 上传到github
  5. 博客就可以在公网访问了
    abc.github.io
  6. 后期-写文章
    写在blog/source/_post下面
    然后 hexo clean - 清除缓存文件
    hexo g - 生成缓存文件
    hexo d - 推送到github
  7. 换主题
    volantis发现一款很不错的主题，只需俩步：
    • 修改配置：theme: volantis
    • blog更目录输入 npm i hexo-theme-volantis
